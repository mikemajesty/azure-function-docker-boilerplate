## Azure function with Docker


### Prerequisite
 - Node 16
 - Docker


### Instalation

```bash
$ npm i
```

### Running the app

```bash
# F5 in VSCODE  
# or
$ npm run start
# or
$ docker-compose up --build
```


### Deploy
```bash
$ npm run deploy:dev
# or
# Use VSCODE extension
```
